Capabilities

A–B section loops

Repeat a passage N times, with optional per-pass speed ramps (0.5× → 0.75× → 1.0×) — built for practicing music or language.

Clip sequences

Play a list of segments back-to-back, skipping the gaps between them. Returns instantly; the sequence runs in the browser.

Named timestamps

Save labeled positions and loopable sections per video, then jump straight to them by name later.

Saved-video library

Build and search a personal library of videos stored locally, and open any of them in a managed tab.

Transcript search

Find a word or phrase in the captions and jump straight to where it’s said — timestamps with context, fetched server-side.

Parallel sessions

Run several agent sessions at once — each drives its own YouTube tab in parallel, playing different videos side by side.

How it works

One broker owns the localhost port and routes each session to its own tab. Every server shares a single library on disk.

Install

Node 24 LTS and Chrome. Two minutes, no build.

  1. Install the server

    One global npm package — no clone, no build. 

    $ npm i -g playback-mcp

  2. Load the extension

    Download playback-mcp-extension-vX.Y.Z.zip from the latest release and unzip it. In chrome://extensions, enable Developer mode, click Load unpacked, and pick the unzipped folder.

  3. Register it with your MCP client

    Any MCP client works. For Claude Code, for example:

    $ claude mcp add playback-mcp -- playback-mcp

    Other clients register a command the same way — point them at playback-mcp. Or use a project-scoped .mcp.json at your repo root — see the README.

  4. Open a video and ask

    Navigate to a YouTube video and ask your agent to control it — “pause it,” “loop the chorus twice at 0.75×,” “find where they mention hooks.”

Local-first & safe

Bot-safe

The extension only reads and writes the <video> element’s own properties — current time, rate, volume, play/pause. It never clicks UI, never scrapes, and never automates navigation beyond opening a watch URL.

Private

Everything runs on 127.0.0.1. Your saved videos and timestamps live in a local SQLite database on your own computer. This extension sends nothing to any external server.
